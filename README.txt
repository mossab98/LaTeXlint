Execution:


You can run the linter directly from the linter.py file from the terminal by inserting the file name of the .tex file you want to have edited. The program will make a new edited of the original .tex file.


JSON:


The rules can be configurable by the user, if you want to activate or deactivate the rules you change the value to true or false. The amount of blankline can also be configured in the same way, just by changing the value in JSON.
